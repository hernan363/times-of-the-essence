- assets/main-logo.png
- assets/divider.png
- assets/clockwork-mark.png
- assets/seal.png
- assets/monogram.png
- assets/vertical-ornament.png

Deploy to S3:
1. Upload index.html and the assets/ folder to your S3 bucket.
2. Enable Static website hosting.
3. Set index document to: index.html
4. Replace contact placeholders in index.html:
   - john@example.com
   - P.O. Box 000 / City, State ZIP

Notes:
- The site intentionally has no cart or payment mechanism.
- The watch photos are stock-photo hotlinks. For a production site, replace the background-image URLs in the CSS with your own licensed stock photos or product photos.
